Tài liệu hướng dẫn xây dựng Website CMS Full-stack bằng ASP.NET Core + ReactJS + SQL Server, chia thành nhiều buổi học từ cơ bản đến nâng cao.

Buổi 1: Khởi tạo dự án
Cài đặt môi trường (Visual Studio, .NET, SQL Server, Node.js).
Tạo Solution gồm 3 project:
CMS.Data (Entity, Database)
CMS.Backend (ASP.NET Core MVC/API)
CMS.Frontend (ReactJS)
Xây dựng các Entity: Category, Post, User, Product, Customer, Order,...
Tạo Controller, View và hiển thị dữ liệu mẫu.
Buổi 2: Entity Framework Core
Cài đặt EF Core.
Tạo ApplicationDbContext.
Cấu hình Connection String.
Thực hiện Migration (Add-Migration, Update-Database).
Kết nối SQL Server và thay dữ liệu giả bằng dữ liệu thật.
Buổi 3: LINQ và CRUD
Truy vấn dữ liệu bằng LINQ (Where, OrderBy, Include, FirstOrDefault).
Hiển thị dữ liệu có liên kết bảng.
Thực hiện đầy đủ chức năng CRUD:
Thêm (Create)
Sửa (Edit)
Xóa (Delete)
Xem chi tiết (Details)
Các buổi tiếp theo
Xây dựng giao diện quản trị bằng ASP.NET Core MVC.
Kiểm tra dữ liệu (Validation).
Đăng nhập và phân quyền bằng Identity.
Xây dựng RESTful API.
ReactJS gọi API bằng Axios.
Điều hướng với React Router.
Hoàn thiện Website CMS Full-stack.
Debug, tối ưu và báo cáo đồ án.
